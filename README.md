# 🚀 i风

这是一个在安卓手机上纯本地、断网运行的极速大语言模型 App。

## ✨ 核心硬核特性
* **极致性能**：基于底层 C++ `llama.cpp` 引擎浅度调教。
* **满血多核**：支持自定义 CPU 线程数与 GPU 加速切换，榨干骁龙/天玑最后一滴算力。
* **丝滑 UI**：全套 Jetpack Compose 打造，包含遵循 3D 物理惯性的惊艳开屏动画，完美适配深/浅色模式。
